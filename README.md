# 🏡 Real Estate Data Science Project

## 📌 Project Overview
This project analyzes real estate price trends using data science techniques.  
We clean, explore, and visualize housing market data to understand **key factors affecting home prices**.

## 📊 Dataset
- **Source:** [Kaggle](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data)  
- **File:** `real_estate_prices.csv`  
- **Features Include:**
  - `SalePrice`: Price of the house
  - `LotArea`: Size of the property in square feet
  - `YearBuilt`: Year the house was built
  - `Bedrooms`: Number of bedrooms
  - `OverallQual`: Overall material and finish quality

## 🛠 Tools Used
- **Python 🐍**: Pandas, NumPy, Matplotlib, Seaborn  
- **Jupyter Notebook** (Google Colab)  
- **GitHub** for version control  

## 📈 Key Findings
✅ **Most expensive homes** tend to have larger lot sizes and higher quality materials.  
✅ **Older homes** don’t always decrease in value; some well-maintained ones appreciate over time.  
✅ **Price Distribution is Right-Skewed**, meaning most homes are affordable, but a few high-end properties increase the median price.
